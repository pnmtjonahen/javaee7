JavaEE7 JAX RS with BASIC Authentication - Additional Information

This is a implementation of the person application as it described in "RESTFul Java WebServices Security" with additional Glasshfish configuration

  • Add a user with correct group to the Glasshfish config

    Open : GlasshFish admin consle, goto Configurations -> Server Config -> Security -> Realms and open the file realm. Add a user with a mapped group

    The mapped groups can be found in the WEB-INF/glasshsfish-web.xml

  • Start SoapUI and perform a post on the service url
